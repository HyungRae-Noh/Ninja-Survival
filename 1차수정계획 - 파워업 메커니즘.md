# 기본 파워업 목록 (Tier 1: 코어 스탯 및 유틸리티) - 공격 성능 섹션

> **참고**: 공격 파워업은 각각 최대 레벨이 정해져 있습니다.

## 공격 성능- 기본 파워업

### 1. 피해량 증가 (Might)

| 항목 | 내용 |
|------|------|
| **핵심 효과** | 모든 공격의 기본 피해량 비율 증가 |
| **게임 디자인 논리** | 가장 기본적인 플레이어의 화력 증강 요소. VS에서 attackDamage 강화는 랭크당 점진적으로 증가하는 핵심 스탯이며, 쿨타임, 투사체 수와 함께 전투력 최우선 강화 요소로 꼽힙니다. |
| **현재 구현된 코드 스니펫** | `attackDamage: 1.0, // 공격력 배수` |
| **최대 레벨** | 10 (점진적으로 증가) |
| **비고** | 플레이어의 기본 세팅 공격력의 최대 100% |

---

### 2. 공격 속도 증가 (Cooldown)

| 항목 | 내용 |
|------|------|
| **핵심 효과** | 무기 발사 간격(쿨타임) 비율 감소 |
| **게임 디자인 논리** | DPS(초당 피해량)를 비약적으로 상승시키는 핵심 스탯. HTML5/JS 게임에서 수리검(Shuriken)의 공격 속도(주기적 이벤트)는 deltaTime을 사용하여 FPS와 독립적으로 실제 시간 기준으로 관리되어야 합니다. |
| **현재 구현된 코드 스니펫** | `attackSpeed: 1.0, // 공격 속도 배수` |
| **최대 레벨** | 10 (점진적으로 증가) |
| **비고** | 플레이어의 기존 공격 속도의 최대 100% |

---

### 3. 투사체 수 증가 (Amount)

| 항목 | 내용 |
|------|------|
| **핵심 효과** | 한 번에 발사되는 투사체 개수 증가 (+1) |
| **게임 디자인 논리** | 광역 커버 능력 강화. VS에서 랭크당 투사체 개수 +1 증가하며, 무기 진화의 핵심 스탯입니다. 이 스탯은 플레이어가 화면을 압도하는 'Bullet Heaven' 경험을 느끼게 하는 데 가장 직관적으로 기여합니다. |
| **현재 구현된 코드 스니펫** | `projectileCount: 1, // 한번에 나가는 투사체 개수` |
| **최대 레벨** | 10 |
| **비고** | 레벨 업마다 투사체(수리검)이 투사체끼리 겹치지 않고 날아감. 최대 투사체: 10개 |

---

### 4. 투사체 관통 수 증가

| 항목 | 내용 |
|------|------|
| **핵심 효과** | 투사체가 관통할 수 있는 적의 수 증가 |
| **게임 디자인 논리** | 다수 적 처리에 유리. 관통 능력이 없는 무기(예: 기본 투사체)에 레이저 빔의 속성을 부분적으로 부여하여 효율을 높일 수 있습니다. |
| **현재 구현된 코드 스니펫** | `penetration: 0, // 관통 수 (정수)` |
| **최대 레벨** | 5 |
| **비고** | 기본 투사체가 최대 5명의 적을 관통할 수 있음 |

---

### 5. 치명타 확률 증가

| 항목 | 내용 |
|------|------|
| **핵심 효과** | 공격 시 치명타(기본 공격력의 150% 데미지) 발생 확률 증가 |
| **게임 디자인 논리** | 피해량 변동성 추가. 이 스탯은 행운(Luck) 스탯과 연계하여 빌드의 잠재적인 피해량을 높이는 역할을 합니다. |
| **현재 구현된 코드 스니펫** | `critChance: 0.0, // 치명타 확률 (0.0 ~ 1.0)` |
| **최대 레벨** | 10 |
| **비고** | 치명타 확률은 최대 레벨 달성 시 100% |

---

### 6. 치명타 데미지 증가

| 항목 | 내용 |
|------|------|
| **핵심 효과** | 치명타 발생 시 데미지 배율 증가 |
| **게임 디자인 논리** | 치명타 빌드의 최종 강화. 치명타 확률과 함께 투자할 때 폭발적인 피해를 기대할 수 있습니다. |
| **현재 구현된 코드 스니펫** | `critDamage: 1.5, // 치명타 데미지 배수 (기본 150%)` |
| **최대 레벨** | 10 |
| **비고** | 치명타 데미지 배수는 최대 레벨 달성 시 300% |

## 방어 및 생존 (Defense and Survival)

### 1. 최대 체력 증가 (Max Health)

| 항목 | 내용 |
|------|------|
| **핵심 효과** | 최대 체력량 비율 증가 (VS: 랭크당 10% 증가) |
| **게임 디자인 논리** | 생존력 강화. VS의 경우 Max Health 강화는 랭크당 최대 체력 10% 증가 효과를 가지며, 플레이어가 적과의 충돌이나 피해를 흡수할 여유를 제공합니다. |
| **현재 구현된 코드 스니펫** | `maxHealth: 1.0, // 최대 체력 배수` |
| **최대 레벨** | 10 |
| **비고** | 최대 레벨 달성 시, 원래 체력의 100% 증가 |

---

### 2. 받는 피해 감소 (%)

| 항목 | 내용 |
|------|------|
| **핵심 효과** | 플레이어가 받는 피해량 비율 감소 |
| **게임 디자인 논리** | 수비적인 능력치 강화. VS의 damageReduction은 플레이어가 적과 피격 시 체력 감소를 줄여줌 |
| **현재 구현된 코드 스니펫** | `damageReduction: 0.0, // 받는 피해 감소 % (0.0 ~ 0.5)` |
| **최대 레벨** | 10 |
| **비고** | 초기 값은 0.0. 최대 레벨 달성 시 받는 피해 감소 수치: 50% |

---

# 개발 가이드
## 메커니즘 : 최대 레벨이 10이라는 것은 레벨 업 시 파워업 선택화면에서 같은 파워업을 최대 10번 선택할 수 있으며(특정 증강 10레벨 달성 시, 레벨업 시 해당 증강은 노출되지 않음), 최대 레벨이 달성됐다면 비고의 최대 수치까지 플레이어의 스탯을 올린다.
## 특수 파워업과 논리적 증강 파워업
특수 파워업은 플레이어의 레벨 10 달성 후 부터 등장하며  해당 특수 파워업을 선택해야 논리적 증강 파워업이 플레이어의 파워업 선택화면에 노출되고 선택 시 적용된다.(예를 들어, 13레벨 달성 후 호카게의 가호를 선택해서 해당 효과를 적용했다면, 14레벨 부터 논리적 증강 파워업인 방어 스택 충전이 일정 확률에 따라 뜬다., 호카게의 가호를 선택하지 않아다면 방어 스택 충전이 뜨지 않는다.)
## "**최대 레벨**" 칸이 "-"로 이루어진 경우 소모성 파워업이며 한번 선택하면 최대 효과가 적용되고 이후에는 선택창에 표시되지 않는다.(체력회복 아이템은 제외 낮은 확률로 계속 노출)
